# TechVision Solutions - Team Project

## Project Overview

This is the starter repository for the WDD 331R Team Project: Multi-File CSS Architecture. Your team will transform this poorly-designed website into a professional, well-organized CSS implementation following CUBE CSS and ITCSS principles.

## Getting Started

1. Clone this repository to your local machine
2. Open the project in your code editor
3. Coordinate with your team to assign tasks
4. Start working on your assigned CSS files
5. Test your changes by opening the HTML files in a browser
6. Commit and push your changes regularly
7. Communicate with your team about progress

## Important Notes

- **DO NOT modify** `main.css`, `reset.css`, or any HTML files
- **DO NOT modify** `theme.js`
- Use modern nested CSS syntax
- Reference CSS custom properties from `colors.css` and `variables.css`
- Test in both light and dark themes
- Communicate with your team regularly

## Deployment

When your team is ready to deploy:

1. Commit all final changes
2. Push to GitHub
3. Enable GitHub Pages in repository settings
4. Select "main" branch as source
5. Share the live URL with your team
